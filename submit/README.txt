Software: python3, jupyter notebook
Python3 libraries:
	numpy:1.15.2 
	matplotlib:3.0.0 
	scikit-learn:0.20.0 
	xgboost:0.80
Data Set(fashion-mnist):
	https://github.com/zalandoresearch/fashion-mnist
How to run:
	Place dataset files and code files in the same directory, use following command to run 
	the code:
		python3 DT.py
		python3 RF.py
		python3 xgboost.py
	or
		jupyter notebook DT.ipynb
		jupyter notebook RF.ipynb
		jupyter notebook xgboost.ipynb 
		and run each cell
